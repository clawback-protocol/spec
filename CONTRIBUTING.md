# Contributing to Clawback Protocol

This is an early-stage open protocol. Contributions welcome — especially from cryptographers, security researchers, and privacy engineers.

## What We Need Most

- **Cryptographic review** — is the PRE simulation sound? edge cases in key derivation?
- **Umbral integration** — replace HKDF simulation with real NuCypher Umbral PRE
- **Rust port** — core crypto primitives in Rust (`ring`, `x25519-dalek`, `chacha20poly1305`)
- **Attack surface analysis** — what does a malicious broker look like?
- **Whitepaper review** — coming soon

## How to Contribute

1. Fork the repo
2. Open an issue first for significant changes
3. Write tests for anything you add
4. Submit a PR with a clear description

## Security Issues

Please do **not** open public issues for security vulnerabilities.  
Email: admin@second2none.net
